# Прогнозирование кредитного дефолта (PD) и стресс-тестирование портфеля

**Проект** по моделированию вероятности дефолта заёмщиков на основе реальных данных Home Credit Default Risk (Kaggle).  
Демонстрирует работу с большими данными (PySpark), построение ML-моделей (LightGBM, XGBoost, CatBoost, нейросеть PyTorch), интерпретацию через SHAP и стресс-тестирование портфеля.

---

## Цели проекта

- Обработать 8 таблиц (анкеты, кредитная история, платежи, балансы) с помощью PySpark.
- Создать более 40 признаков (долг/доход, просрочки, агрегаты по предыдущим заявкам и т.д.).
- Сравнить классические бустинги и нейросеть.
- Интерпретировать лучшую модель с помощью SHAP.
- Провести стресс-тестирование: оценить рост риска при макроэкономическом шоке.

1. **Клонировать репозиторий**

## API для предсказаний (FastAPI)

Для интеграции модели в приложения разработан микросервис на FastAPI.

**Эндпоинты**:
- `GET /health` – проверка работоспособности.
- `POST /predict` – получение вероятности дефолта для одного клиента.

**Пример запроса**:
```json
{
  "EXT_SOURCE_2": 0.6,
  "EXT_SOURCE_3": 0.7,
  "AMT_CREDIT": 500000,
  "DEBT_TO_INCOME": 0.3,
  "EMPLOYMENT_LEN": 5,
  "AGE": 37
}
```

**Запуск API**:
```bash
uvicorn api.main:app --reload
```

После запуска документация доступна по адресу http://127.0.0.1:8000/docs.

## Основные результаты

### Модели (F1 macro)

| Модель                     | F1    | Порог |
|----------------------------|-------|-------|
| LightGBM                   | 0.479 | 0.20  |
| XGBoost                    | 0.625 | 0.20  |
| CatBoost                   | 0.627 | 0.70  |
| Ансамбль (XGB+CatBoost)    | 0.628 | 0.44  |
| Нейросеть (PyTorch)        | 0.595 | 0.50  |

**Лучшая модель** – CatBoost (F1=0.627).

### SHAP (CatBoost)

- Топ-5 важных признаков: `EXT_SOURCE_2`, `EXT_SOURCE_3`, `POS_CNT_RECORDS`, `AMT_GOODS_PRICE`, `EXT_SOURCE_1`.
- Высокие значения `EXT_SOURCE_2/3` снижают риск, высокий `DEBT_TO_INCOME` и `AMT_CREDIT` – повышают.
- Нелинейность `EXT_SOURCE_2`: SHAP положителен при <0.3, отрицателен при >0.6.

### Стресс-тестирование

**Сценарий**: ↓ дохода на 20%, стаж = 0, рейтинг региона → 3, ↓ внешних скорингов на 10%.

| Метрика                               | Базовый | Стресс | Изменение |
|---------------------------------------|---------|--------|------------|
| Средняя вероятность дефолта (PD)      | 0.384   | 0.472  | +22.9%     |
| Доля клиентов с PD > 0.15             | 87.3%   | 96.7%  | +9.4 п.п.  |

**Вывод**: портфель чувствителен к кризисным шокам.
